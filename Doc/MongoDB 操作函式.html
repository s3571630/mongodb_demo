以下是所有 MongoDB 聚合框架中的常用 $ 操作符，已按照分類進行整理，包括比較、邏輯、數學、字符串、數組、日期、條件、數據類型等操作符。這份清單應涵蓋了 MongoDB 中常用的所有聚合操作符。

操作符說明語法和示例
$eq等於{ field: { $eq: value } } 例如：{ age: { $eq: 30 } }
$ne不等於{ field: { $ne: value } } 例如：{ status: { $ne: "A" } }
$gt大於{ field: { $gt: value } } 例如：{ price: { $gt: 100 } }
$gte大於或等於{ field: { $gte: value } } 例如：{ age: { $gte: 18 } }
$lt小於{ field: { $lt: value } } 例如：{ price: { $lt: 50 } }
$lte小於或等於{ field: { $lte: value } } 例如：{ stock: { $lte: 20 } }
$in在指定數組中{ field: { $in: [ value1, value2, ... ] } } 例如：{ category: { $in: ["Electronics"] } }
$nin不在指定數組中{ field: { $nin: [ value1, value2, ... ] } } 例如：{ status: { $nin: ["B", "C"] } }

2. 邏輯操作符（Logical Operators）

操作符說明語法和示例
$and邏輯與，所有條件必須為真{ $and: [ { condition1 }, { condition2 }, ... ] } 例如：{ $and: [ { price: { $gt: 100 } }, { stock: { $lte: 50 } } ] }
$or邏輯或，至少有一個條件為真{ $or: [ { condition1 }, { condition2 }, ... ] } 例如：{ $or: [ { status: "A" }, { quantity: { $lt: 20 } } ] }
$not邏輯非，反轉指定條件的布爾值{ field: { $not: { condition } } } 例如：{ price: { $not: { $gt: 100 } } }
$nor邏輯非或，所有條件必須為假{ $nor: [ { condition1 }, { condition2 }, ... ] } 例如：{ $nor: [ { status: "A" }, { qty: { $lt: 20 } } ] }

3. 數學操作符（Arithmetic Operators）

操作符說明語法和示例
$add將兩個或更多數字相加{ $add: [ <expression1>, <expression2>, ... ] } 例如：{ total: { $add: ["$price", "$tax"] } }
$subtract從第一個數字減去第二個數字{ $subtract: [ <expression1>, <expression2> ] } 例如：{ difference: { $subtract: ["$price", "$discount"] } }
$multiply將兩個或更多數字相乘{ $multiply: [ <expression1>, <expression2>, ... ] } 例如：{ total: { $multiply: ["$price", "$quantity"] } }
$divide將第一個數字除以第二個數字{ $divide: [ <expression1>, <expression2> ] } 例如：{ ratio: { $divide: ["$total", "$count"] } }
$mod返回第一個數字除以第二個數字的餘數{ $mod: [ <expression1>, <expression2> ] } 例如：{ remainder: { $mod: ["$total", 100] } }

4. 字符串操作符（String Operators）

操作符說明語法和示例
$concat將兩個或多個字符串拼接在一起{ $concat: [ <expression1>, <expression2>, ... ] } 例如：{ fullName: { $concat: ["$first_name", " ", "$last_name"] } }
$substr返回指定位置的子字符串{ $substr: [ <string>, <start>, <length> ] } 例如：{ shortName: { $substr: ["$name", 0, 3] } }
$toLower將字符串轉換為小寫{ $toLower: <string> } 例如：{ lowerCaseName: { $toLower: "$name" } }
$toUpper將字符串轉換為大寫{ $toUpper: <string> } 例如：{ upperCaseName: { $toUpper: "$name" } }
$regexMatch使用正則表達式匹配字符串{ $regexMatch: { input: "$name", regex: "^J" } }

5. 數組操作符（Array Operators）

操作符說明語法和示例
$size返回數組的大小（長度）{ $size: <array> } 例如：{ itemCount: { $size: "$items" } }
$slice返回數組的子集{ $slice: [ <array>, <position>, <n> ] } 例如：{ topItems: { $slice: ["$items", 0, 3] } }
$arrayElemAt返回數組中指定索引位置的元素{ $arrayElemAt: [ <array>, <idx> ] } 例如：{ firstItem: { $arrayElemAt: ["$items", 0] } }
$in檢查值是否存在於數組中{ <field>: { $in: [ <value1>, <value2>, ... ] } } 例如：{ status: { $in: ["A", "B"] } }

6. 日期操作符（Date Operators）

操作符說明語法和示例
$dateToString將日期轉換為字符串{ $dateToString: { format: "%Y-%m-%d", date: "$dateField" } }
$year返回日期的年份{ $year: <date> } 例如：{ year: { $year: "$dateField" } }
$month返回日期的月份{ $month: <date> } 例如：{ month: { $month: "$dateField" } }
$dayOfMonth返回日期的日{ $dayOfMonth: <date> } 例如：{ day: { $dayOfMonth: "$dateField" } }
$hour返回日期的小時{ $hour: <date> } 例如：{ hour: { $hour: "$dateField" } }
$minute返回日期的分鐘{ $minute: <date> } 例如：{ minute: { $minute: "$dateField" } }
$second返回日期的秒{ $second: <date> } 例如：{ second: { $second: "$dateField" } }

7. 條件操作符（Conditional Operators）

操作符說明語法和示例
$cond基於條件返回不同的值{ $cond: { if: <expression>, then: <expression>, else: <expression> } } 例如：{ $cond: { if: { $gt: ["$quantity", 10] }, then: "big", else: "small" } }
$ifNull如果第一個參數為 null，則返回第二個參數{ $ifNull: [ <expression>, <replacement-if-null> ] } 例如：{ name: { $ifNull: ["$username", "unknown"] } }

8. 類型操作符（Type Operators）

操作符說明語法和示例
$type返回字段的 BSON 類型{ $type: <field> } 例如：{ fieldType: { $type: "$age" } }
$exists測試字段是否存在{ <field>: { $exists: <boolean> } } 例如：{ age: { $exists: true } }

9. 聚合管道階段操作符（Pipeline Stage Operators）

操作符說明語法和示例
$match過濾文件，只傳遞符合指定條件的文件{ $match: { status: "A" } }
$group將輸入文件按指定的 _id 表達式分組{ $group: { _id: "$user_id", totalAmount: { $sum: "$amount" } } }
$project改變輸出文件的結構，只保留指定的字段{ $project: { item: 1, totalAmount: 1 } }
$sort根據一個或多個字段對輸出文件進行排序{ $sort: { totalAmount: -1 } }
$limit限制傳遞到下一階段的文件數量{ $limit: 5 }
$skip跳過指定數量的文件，傳遞剩餘的文件{ $skip: 10 }
$lookup執行左外連接（left outer join）以從另一個集合中引入文件{ $lookup: { from: "orders", localField: "user_id", foreignField: "user_id", as: "orders" } }
$unwind將每個輸入文件中的某個數組字段展開為多個文件{ $unwind: "$items" }
$addFields添加新字段或重命名現有字段{ $addFields: { totalWithTax: { $multiply: ["$total", 1.1] } } }
$out將聚合管道的結果寫入指定的集合{ $out: "aggregated_results" }
$merge將管道的結果合併到現有集合中{ $merge: { into: "myCollection", on: "_id", whenMatched: "merge", whenNotMatched: "insert" } }

10. 其他操作符

操作符說明語法和示例
$geoNear返回按距離排序的文件{ $geoNear: { near: { type: "Point", coordinates: [ <lng>, <lat> ] }, distanceField: "dist.calculated", spherical: true } }
$text執行文本搜索{ $text: { $search: "keyword" } }
$redact基於文檔級安全性來限制返回的字段{ $redact: { $cond: { if: <condition>, then: <action>, else: <action> } } }
